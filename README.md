# Framework - Svelte

Projeto utilizando o framework svelte.

O objetivo desse repositório é criar um website de raças de cachoro estilo google para colocar em prática conhecimentos sobre Svelte.

### Definição do Svelte

O Svelte é um compilador de front-end inovador, criado por Rich Harris em 2016, que transforma componentes em JavaScript puro e otimizado, sem utilizar Virtual DOM.

### Porque utilizá-lo

O Svelte torna todo o código em JavaScipt puro, o que torna a aplicação mais rápida e com pacotes menores.

### Indicações e Contra-Indicações

O Svelte não é indicado para empresas ou projetos grandes, devido ao seu ecossistema ser menor.

 Mas em compensação ele é o melhor framework para se usar em projetos pequenos e médios, pois tem uma curva de aprendizagem rápida, código mais limpo e legível e performance no cliente.

 ## Instalação e Configuração

 Para criar um novo projeto baseado neste modelo:

 ``` 
npx degit sveltejs/template svelte-app
cd svelte-app 
```

Agora se deve instalar as dependencias... 

 ``` 
cd svelte-app
npm install
```

E rodar o projeto...

 ``` 
npm run dev
```

Acesse [localhost:8080](http://localhost:8080/) . Você deverá ver seu aplicativo em execução.
